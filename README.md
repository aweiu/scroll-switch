# scroll-switch

简单来说，它就是用来滚动切换列表项的。

## 特性

- next、pre 会根据当前列表滚动位置智能切换到最符合用户预期的条目
- 相对于 [scrollIntoView](https://developer.mozilla.org/zh-CN/docs/Web/API/Element/scrollIntoView) 支持设置 offset，可以更好的控制滚动目标位置
